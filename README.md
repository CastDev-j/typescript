# TypeScript + Bun Starter

Un starter básico para proyectos con TypeScript y Bun runtime.

## Requisitos

- [Bun](https://bun.sh) instalado

## Instalación

```bash
bun install
```

## Desarrollo

Para ejecutar el código en modo desarrollo:

```bash
bun run dev
```

## Compilar

Para compilar TypeScript a JavaScript:

```bash
bun run build
```

## Ejecutar

Para ejecutar el proyecto compilado:

```bash
bun run start
```

## Testing

Para ejecutar los tests:

```bash
bun test
```

## Estructura

```
├── src/
│   ├── index.ts       # Archivo principal
│   └── hello.test.ts  # Tests
├── dist/              # Output compilado
├── package.json
├── tsconfig.json
└── README.md
```
